# DL2416T Pico-SDK Library

This library is a basic driver for the Siemens DL2416T display. It should also be compatible with later pin compatible models, such as the DLG2416 and DLR2416, but this is as of yet untested.

An example implementaiton of this library can be found in the microclock_sw repo: https://github.com/davidb990/microclock_sw
